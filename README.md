## Overview

The **Mafalia Partner Network** platform empowers partners to efficiently grow their network. It provides a unified interface for enrolling both individual clients (Mafalia) and business clients (Commercia) across West Africa, while offering real-time analytics and educational resources.

## Features

### Dashboard & Analytics
- **Real-time Statistics**: Track active clients, revenue, and network growth.
- **Performance Charts**: Visualize monthly and yearly performance trends.
- **Commission Tracking**: Monitor earnings with a detailed commission widget.
- **Leaderboard**: Compete with other partners and track rankings.

### Client Enrollment
- **Unified Interface**: Enroll new clients for **Mafalia** (Individuals) or **Commercia** (Businesses).
- **Multi-Country Support**: Optimized for Senegal, Mali, Côte d'Ivoire, Burkina Faso, and Guinea.
- **Regions & Cities**: Dynamic form selection for accurate client data.

### Sales Tools
- **Commercial Scripts**: Access proven sales pitches and arguments.
- **Marketing Materials**: Download official brochures and visual aids.
- **Virtual Presentations**: Watch tutorials and product demonstrations directly in the app.

## Technology Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Directory)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide Icons](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Language**: TypeScript

## Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/utachicodes/mafalia-mtn.git
   cd mafalia-partner-network
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/dashboard`: Main dashboard layout and pages.
- `app/dashboard/enrolement`: Client enrollment forms.
- `app/dashboard/sales-tools`: Sales scripts and brochures.
- `app/dashboard/presentations`: Video tutorials and presentations.
- `components/ui`: Reusable UI components.
